This is the API

createNewPatient

Create a new patient in the system

URL

https://virtserver.swaggerhub.com/Beam_Health/Beam_API/1.0.0/v0/practices/{practice_id}/staff/{staff_id}/patients/

HTTP Method

POST

Response Type

Parameters

  • Body: body Patient(Patient)

    Patient Info

  • Path: staff_id Integer

    Unique ID of the staff member

  • Path: practice_id Integer

    Unique ID of the staff practice

deletePatient

Deletes an existing patient in the system

URL

https://virtserver.swaggerhub.com/Beam_Health/Beam_API/1.0.0/v0/practices/{practice_id}/staff/{staff_id}/patients/{patient_id}/

HTTP Method

DELETE

Response Type

Parameters

  • Path: staff_id Integer

    Unique ID of the staff member

  • Path: practice_id Integer

    Unique ID of the staff practice

  • Path: patient_id String

    Unique full ID of the patient
